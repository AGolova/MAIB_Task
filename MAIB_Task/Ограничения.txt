Ограничения:
1. У пользователя должны быть права на полный доступ к БД tempbd
2. В папке должны быть только файлы для импорта
3. Все данные внутри файла должны быть заключены в "" и разделитель табуляция \t
4. Название таблиц для импорта данных и название процедуры должны сопровождаться комментом
5. Из процедуры, должна возвращаться 1 строка текста каждого отчета,того что должно быть в html файле
 
Плюсы по сравнению с предыдущей версией:
сервер может быть удаленный
Расширение файла не имеет значение.
Структура файла- кол-во колонок неважна, если в процедуре таблица имеет такой же вид.
процедура может возвращать N количество отчетов

Изменения:
из процедуры убраны все недокументированные функции, которые работали с файлами.
появилось окно при обработке данных
возможность работать по принципу клиент-сервер(удаленный)
